          /games/delete delete

          delete

          Deletes either a game or a turn from the database. If the user only specifies a GID then the whole game will be deleted. If the user also includes a valid turn in their request, the API will only delete the specific turn.

          delete /games/delete

          Deletes either a game or a turn from the database. If the user only specifies a GID then the whole game will be deleted. If the user also includes a valid turn in their request, the API will only delete the specific turn.

          • Request
          • Response

          Query Parameters

          • gid: required(string)

            Game ID

            Example:

            t_32
          • turn: (string)

            Turn number to delete. This is optional.

            Example:

            2

          HTTP status code 200

          /games/snake post

          post

          This endpoint should not be called directly, let the Javascript front end do it. That said the function of this endpoint is to make an API call to the battlesnake. It is put here because CORS would block it if the client were to directly call the battlesnake.
